## 5. 운영(호스팅) — axdev 서버에 도메인 연결
정적 사이트라 가장 단순. 같은 서버에 **독립 nginx 가상호스트**로 운영:
1. `index.html`+`css/`+`images/`+`img/` 를 서버 `/var/www/poesia/` 배치(`index_original.html`·`README_*`·`*.bak`·`*.php` 제외).
2. DNS: `smilewithpoesia.com`(A)·`www`(A/CNAME) → 서버 공인 IP. **MX(이메일) 레코드는 유지**.
3. nginx server_name 블록 추가(axdevhub 앱과 분리) + `www`→apex 301.
4. certbot(Let's Encrypt) HTTPS.
- 주의: 현재 도메인이 타 호스팅 중이면 TTL 낮추고 서브도메인 선검증 후 전환.
